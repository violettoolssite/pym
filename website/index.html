PVM

跨平台 Python 版本管理工具，灵感来自 nvm。 一键安装、切换和管理多个 Python 版本。 支持 Windows (CMD/PowerShell)、Linux 和 macOS。

Windows Linux macOS CMD PowerShell Bash
Quick Install GitHub

Quick Install

Windows (PowerShell)

irm https://pvm.violetteam.cloud/install.ps1 | iex

以管理员身份运行 PowerShell

Linux / macOS

curl -fsSL https://pvm.violetteam.cloud/install.sh | bash

需要 curl 和构建工具

China Mirror / 国内镜像

irm https://gh.llkk.cc/https://raw.githubusercontent.com/violettoolssite/pym/main/install.ps1 | iex
curl -fsSL https://gh.llkk.cc/https://raw.githubusercontent.com/violettoolssite/pym/main/install.sh | bash

Usage / 使用方法

Terminal 
# List available Python versions
$ pvm list available

# Install Python 3.12.4
$ pvm install 3.12.4

# Switch to Python 3.12.4
$ pvm use 3.12.4

# Show current version
$ pvm current
Current Python version: 3.12.4

# List installed versions
$ pvm list
  * 3.12.4 (current)
    3.11.9
    3.10.14

# Uninstall a version
$ pvm uninstall 3.10.14

Features / 功能特性

Quick Install

一键安装多个 Python 版本，自动下载和配置，无需手动设置环境变量。

Easy Switch

一条命令切换 Python 版本，自动更新 PATH，立即生效。

Cross Platform

支持 Windows (CMD/PowerShell)、Linux 和 macOS，统一的命令体验。

Mirror Support

支持配置国内镜像源（华为云、npmmirror），加速下载。

Lightweight

纯脚本实现，无需安装额外依赖，Windows 使用嵌入式 Python。

pip Included

自动安装 pip，开箱即用，立即开始你的 Python 项目。

Mirror Configuration / 镜像配置

使用 pvm config 命令快速配置镜像源：

Terminal 
# Use Tsinghua mirror (recommended for China)
$ pvm config tsinghua

# Use Huawei Cloud mirror
$ pvm config huawei

# Show current config
$ pvm config
Recommended for China

Tsinghua / 清华

pvm config tsinghua

Huawei Cloud / 华为云

pvm config huawei

Aliyun / 阿里云

pvm config aliyun

npmmirror

pvm config npmmirror

Default / 官方

pvm config default
